# Smart Password Generator

Smart Password Generator is a web application that generates strong and memorable passwords based on a user-provided reminder.

Instead of creating completely random passwords, the application transforms a reminder into a stronger password by applying character substitutions and adding extra security elements while keeping the password easy to remember.

---

## Features

- Generate passwords from a personal reminder
- Keep passwords memorable
- Replace common characters with stronger alternatives
- Add random symbols
- Add random numbers
- Generate unique passwords every time

---

## Example

Reminder:

```
Harvard Dream
```

Generated Password:

```
H@rv4rdDr3am#482Q
```

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## How to Use

1. Open the application.
2. Enter a reminder.
3. Click **Generate Password**.
4. Copy and use the generated password.
